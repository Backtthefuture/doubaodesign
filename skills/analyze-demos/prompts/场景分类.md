# 场景分类提示词

## 任务
根据 Demo 的 Prompt 描述和预览内容，判断该 Demo 最适合归类到哪个场景。

## 可选场景（必须从以下18个中选择一个，严格使用以下名称）

| 场景 | 判断依据 |
|------|----------|
| 一句话生成App | 生成完整应用程序，如播放器、编辑器、社交App |
| 一句话做网站 | 生成网页、Landing页面、个人网站 |
| 一句话造游戏 | 生成可玩的游戏，如台球、马里奥、打字游戏 |
| 一句话出3D | 生成3D可视化、3D场景、体素模型 |
| 一句话写系统 | 生成操作系统模拟、系统级程序 |
| 一句话搞特效 | 生成视觉特效，如海浪、烟花、粒子效果 |
| 图生万物 | 输入图片，转换为其他形式（代码、SVG、漫画）|
| 视频秒懂 | 分析视频内容，提取信息或总结 |
| 文档秒读 | 处理文档、PDF、手写内容 |
| AI替你干活 | 自动化操作电脑、Agent能力展示 |
| 一句话出图 | 生成图像、ASCII艺术、体素艺术 |
| 一句话做PPT | 生成演示文稿 |
| 一句话做表格 | 生成数据模型、图表、表格 |
| 写作助手 | 生成文案、文章、话术 |
| AI当老师 | 教学、答疑、辅导功能 |
| AI陪聊 | 情感对话、深度讨论 |
| 视觉考AI | 测试AI视觉理解能力 |
| 奇葩挑战 | 刁钻问题、极限测试 |

## 场景名称映射（如果输入已有场景名但不在上述列表，请映射到正确名称）
- "看图知一切" → "图生万物"
- "图片理解" → "图生万物"
- "视频理解" → "视频秒懂"
- "文档处理" → "文档秒读"
- "Agent" → "AI替你干活"
- "代码生成" → "一句话生成App"

## 输出格式
仅输出场景名称（必须是上述18个之一），不要任何解释。

## 示例

**输入**: "读取VRM启动摄像机，面部捕捉+遮罩+遮挡剔除"
**输出**: 一句话生成App

**输入**: "核电厂工作原理交互式3D模拟器"
**输出**: 一句话出3D

**输入**: "比赛分析、网课总结、监控视频分析"
**输出**: 视频秒懂
