# 模型能力识别提示词

## 任务
根据 Demo 的 Prompt 描述和预览内容，识别该 Demo 展示了哪些模型能力。可以选择1-3个最相关的能力。

## 可选能力（从以下10个中选择）

| 能力标签 | 判断依据 |
|----------|----------|
| 多模态输入 | 能看图、看视频、读文档，处理非文本输入 |
| 代码狂魔 | 生成可运行的代码，如网页、应用、游戏 |
| 3D空间感 | 理解和生成3D场景、空间可视化 |
| 物理引擎脑 | 模拟物理规律，如碰撞、弹跳、流体 |
| Agent自动化 | 自动操作电脑、调用工具完成任务 |
| 超长待机 | 处理超长内容，如长视频、大量文档 |
| 工具达人 | 调用API、外部工具、集成服务 |
| 指令遵循王 | 完美执行复杂指令，生成符合要求的完整内容 |
| 图像生成 | 生成图像、艺术作品 |
| 深度推理 | 复杂问题分析、决策推理 |

## 选择原则
1. 优先选择最突出的能力（1-2个核心能力）
2. 如果明显涉及多个能力，最多选3个
3. "代码狂魔"是最常见的能力，但不要滥用

## 输出格式
以 JSON 数组格式输出能力标签，不要任何解释。

## 示例

**输入**: "读取VRM启动摄像机，面部捕捉+遮罩+遮挡剔除"
**输出**: ["多模态输入", "代码狂魔"]

**输入**: "核电厂工作原理交互式3D模拟器"
**输出**: ["3D空间感", "物理引擎脑"]

**输入**: "收件箱整理、剪映操作、表单填写"
**输出**: ["Agent自动化", "工具达人"]

**输入**: "2小时网课总结"
**输出**: ["多模态输入", "超长待机"]
